# 任务便签 (Sticky Notes)

一个轻量级的桌面悬浮窗任务便签应用，支持置顶、拖拽、折叠、系统托盘与数据持久化。

![Screenshot](1.png)

## 特性

- **始终置顶** — 悬浮窗始终显示在最上层，不被其他窗口遮挡
- **拖拽 & 边缘吸附** — 按住顶部可拖动到屏幕任意位置，释放时自动贴边
- **折叠 / 展开** — 双击标题栏或点击 `-` 按钮，窗口折叠为仅标题栏
- **系统托盘** — 关闭窗口后最小化至托盘，托盘图标支持右键菜单
- **透明度调节** — 托盘右键菜单可调整窗体透明度（80% / 90% / 100%）
- **数据持久化** — 任务与窗口位置自动保存至本地 JSON，重启后完整恢复
- **拖拽排序** — 拖拽任务项可调整顺序，滚动区边缘自动加速滚动
- **圆角裁剪** — 使用 `setMask` 实现完美圆角，无平台兼容问题
- **AI 任务整理** — 右键任务项打开 AI 聊天面板，流式对话整理任务，支持多轮追问
- **多级任务层级** — 支持 1~4 级任务树，子任务可展开/收起，缩进显示

## 快速开始

```bash
pip install PyQt6 aiohttp
python main.py
```

## 使用方法

| 操作 | 说明 |
|---|---|
| 添加任务 | 底部输入框输入文字，回车或点击 `+` 按钮 |
| 标记完成 | 点击任务左侧圆形复选框 |
| 任务详情 | 双击任务项，弹出详情悬浮窗查看完整文本 |
| 删除任务 | 鼠标悬停任务项点击 `×`，或详情窗中点击"删除任务" |
| 拖拽排序 | 按住任务项拖拽可调整顺序，滚动区边缘自动滚动 |
| 折叠窗口 | 双击标题栏，或点击顶部 `-` 按钮 |
| 最小化到托盘 | 点击顶部 `×` 按钮，或关闭窗口 |
| 恢复窗口 | 系统托盘图标双击，或右键菜单选择"显示主窗体" |
| 退出应用 | 右键托盘图标，选择"退出应用" |
| AI 设置 | 托盘右键菜单选择"AI 设置"，配置 API 地址、Key、模型 |
| AI 整理任务 | 右键点击任务项，选择"AI 整理任务"，打开聊天面板 |

## AI 任务整理

右键任意任务选择 **AI 整理任务**，弹出浮动聊天面板：

1. **自动预填** — 当前任务列表自动填入输入框，可编辑后发送
2. **多轮对话** — 与 AI 讨论如何拆分、合并、重组任务
3. **流式输出** — AI 回复逐字流式显示，无需等待完整响应
4. **写入任务** — 点击底部"写入任务"按钮，AI 将讨论内容整理为 JSON 数组
5. **确认后写入** — 弹窗确认后追加到任务列表，支持层级缩进显示

> 首次使用需在 **AI 设置** 中配置 OpenAI 兼容 API（如 OpenAI、DeepSeek、Ollama 等）。

## 数据存储

任务数据保存在系统应用数据目录：

| 平台 | 路径 |
|---|---|
| Windows | `%APPDATA%\StickyNotes\tasks.json` |
| macOS | `~/Library/Application Support/StickyNotes/tasks.json` |
| Linux | `~/.local/share/StickyNotes/tasks.json` |

启动时若文件损坏，会自动备份为 `tasks.json.bak` 并重置为空状态。

## 任务数据结构

```json
{
  "id": 1715000000000,
  "text": "完成项目搭建",
  "completed": false,
  "createdAt": 1715000000000,
  "level": 1,
  "parentId": null,
  "collapsed": false
}
```

| 字段 | 说明 |
|---|---|
| `level` | 任务层级（1~4），控制缩进显示 |
| `parentId` | 父任务 ID，`null` 表示顶级任务 |
| `collapsed` | 是否收起子任务 |

## 技术栈

- **Python 3.10+**
- **PyQt6** — 跨平台 GUI 框架
- **aiohttp** — 异步 HTTP 客户端（AI 流式请求）

## 项目结构

| 文件 | 说明 |
|---|---|
| `main.py` | 主入口：GUI 窗口、任务操作、数据持久化 |
| `ai_organizer.py` | AI API 客户端：异步流式调用、连接测试 |
| `ai_chat.py` | AI 聊天面板：浮动对话、流式输出、多轮交互 |
| `ai_async_worker.py` | Qt ↔ asyncio 桥接：QThread 内运行异步事件循环 |
| `ai_dialogs.py` | AI 配置弹窗、整理输入弹窗、结果预览弹窗 |
| `task_tree.py` | 树形工具函数：构建/展平/可见性过滤/层级分配 |
| `tree_widgets.py` | 层级任务组件：缩进 + 展开/收起 + 右键菜单 |
